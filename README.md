# 💰 Expense Tracker System

### Personal Finance Management Web Application

A full-stack web application designed to help users manage personal finances efficiently by tracking income, expenses, balances, and monthly spending patterns. The system provides secure authentication, transaction management, financial reports, and interactive dashboards to improve financial awareness and budgeting.

---

## 🌟 Overview

Managing personal finances effectively is essential for maintaining financial stability and achieving long-term goals.

The Expense Tracker System enables users to record income and expenses, categorize transactions, monitor financial activities, and generate reports through a user-friendly dashboard.

The application is built using PHP and MySQL and follows software engineering principles, including UML modeling, ER design, and layered architecture.

---

## 🎯 Project Objectives

* Provide a simple and intuitive platform for managing personal finances.
* Allow users to record daily income and expenses.
* Categorize transactions for better organization.
* Automatically calculate balances and financial summaries.
* Generate monthly financial reports and spending insights.
* Secure user data using session-based authentication.
* Support transaction editing, updating, and deletion.
* Deliver a responsive experience across devices.
* Promote financial discipline through expense monitoring and visualization.

## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/AlajingiGanesh/Expense-Tracker-System.git
```

### Move Project

Copy project folder into:

```text
xampp/htdocs/
```

### Start Services

Open XAMPP and start:

```text
Apache
MySQL
```

### Import Database

1. Open phpMyAdmin
2. Create database:

```sql
dailyexpense
```

3. Import:

```text
dailyexpense.sql
```

### Run Application

Open browser:

```text
http://localhost/Expense-Tracker-System
```
